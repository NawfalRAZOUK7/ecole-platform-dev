🔗 Strategie Cross-Platform

PrincipeImplementation
Mobile-First (Eleve + Parent)L'app Flutter est la plateforme principale. Le web offre un acces complementaire avec bridge cards.
Web-First (Enseignant + Admin)Le web React est la plateforme principale pour les taches de gestion. Le mobile offre un acces simplifie.
PlatformBridgeCardComposant reutilisable (React + Flutter) informant l'utilisateur que la fonctionnalite est disponible sur l'autre plateforme.
