# Meta-Prompts

AI execution meta-prompts implementing a 3-phase structure (Analyze, Execute, Verify) for systematic development work.

## Core 3-Phase Prompts

- **META_PROMPT_1_CONTEXT.md** - Phase 1: Context analysis and understanding
- **META_PROMPT_2_EXECUTE.md** - Phase 2: Execution and implementation
- **META_PROMPT_3_VERIFY.md** - Phase 3: Verification and testing

## Specialized Meta-Prompts

- **META_PROMPT_FULL_REFACTOR.md** - Full refactoring meta-prompt
- **META_PROMPT_OOP_1_CONTEXT.md** - OOP refactoring Phase 1: Context
- **META_PROMPT_OOP_2_EXECUTE.md** - OOP refactoring Phase 2: Execution
- **META_PROMPT_OOP_3_VERIFY.md** - OOP refactoring Phase 3: Verification

These meta-prompts guide AI agents through complex multi-step development tasks with consistent structure and validation.
